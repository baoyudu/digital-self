# Digital Self — Export

Generate a clean export of the digital self, optimized for use as context in other AI systems.

## Step 1: Load

Read `me.md` and all dimension files.

## Step 2: Generate export

Create a self-contained markdown document based on `me.md` with these modifications:

1. **Remove internal references.** Strip the "Learn More" section and any links to other files — the export must stand alone.
2. **Expand key details.** If `me.md` references something briefly that is richly detailed in a dimension file, pull in the most essential details to make the export more complete.
3. **Keep it concise.** Target ~800 words (English) / ~2400 characters (Chinese). Longer than `me.md` alone since it has no supporting files, but still compact.
4. **Add a header note:**

```markdown
<!-- This is an auto-generated profile. It can be used as context for AI assistants
     to understand who I am. Generated by digital-self on {date}. -->
```

5. **Ensure the "When Interacting With Me" section is preserved and prominent.** This is the most actionable part for other agents.

## Step 3: Present to user

Show the full export and ask:

> "Here's the exported version. You can paste this into any AI assistant's context or system prompt. Want me to adjust anything before you use it?"

If the user requests changes, make them and show the updated version.

## Step 4: Save (optional)

If the user wants to save it to a file, write it to `{BASE_PATH}/export.md`. This file is separate from `me.md` and can be regenerated anytime.
